# PT Amsar Dashboard

Dashboard manajemen proyek konstruksi untuk PT Amsar Medical Services.

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Recharts
- React Router v6
- jsPDF + html2canvas
- Zustand
- React Hot Toast

## Fitur

- Dashboard KPI proyek
- Manajemen proyek (tambah, filter, tandai selesai)
- Detail proyek: material terpasang, dokumen, foto progres, history aktivitas
- Filter proyek: status, lokasi, bulan/tahun deadline
- Export laporan PDF
- Activity log

## Cara Jalankan

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
